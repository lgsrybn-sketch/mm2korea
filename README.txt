RoVALUES — Netlify-ready build

DEPLOY
1. Open the mm2-korea project in Netlify.
2. Open Deploys and drag this whole folder into the manual deploy area.
3. Keep index.html and the assets folder at the same level.

SERVER LINK
Open index.html and find:
const LINKS = {
  korean: "ВСТАВЬ_ССЫЛКУ_НА_KOREAN_SERVER"
};
Replace the placeholder with the official Roblox private-server URL.
